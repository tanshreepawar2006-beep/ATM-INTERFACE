# ATM Interface (Java)

This is a simple console-based ATM system made using Java.  
The user first enters a User ID and PIN. After a successful login, the ATM menu is shown.

## Features
- Transaction History  
- Withdraw Money  
- Deposit Money  
- Transfer Money  
- Quit  

## Project Structure
ATM INTERFACE  
│── .vscode  
│── bin  
│── lib  
└── src  
   └── ATMInterface.java  

## How to Run
1. Open the project folder in VS Code  
2. Go to the src folder  
3. Compile:javac ATMInterface.java
4. Run:java ATMInterface
